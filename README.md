각 날짜마다 별도의 마크다운(.md) 파일을 생성합니다. 

각 날짜별 파일을 만드는 과정은 다음과 같습니다
- GitHub 저장소에서 'Add file' > 'Create new file' 클릭
- 파일 이름을 '2024-08-01.md'와 같이 입력
- 파일 내용에 그 날의 TIL 내용을 작성
- 'Commit changes file'을 클릭하여 저장
